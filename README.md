# portfolio
임효정 포트폴리오<br><br>
1. 신라호텔 페이지 <br>
신라호텔을 리디자인하고 반응형으로 메인페이지, 서브페이지2개 작업을 하였으며, <br>
아이디 비밀번호가 정해진 값을 입력하였을 때 로그인되었다는 창, 알맞은 값을 입력하지 않으면 아이디 비밀번호를 확인하라는 창이 뜨는 작업까지 완료하였다. 
<br><br>
2. 스킨큐어 (카페24) <br>
카페24를 활용하여 스킨큐어 페이지 작업을 하였다.
<br><br>
3. 부쉐론 (부트스트랩) <br>
부트스트랩을 활용하여 부쉐론 페이지를 리디자인하여 작업하였다.
